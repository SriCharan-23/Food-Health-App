# 🥗 NutriAI – Smart Food & Health Assistant

NutriAI is an intelligent **Flask-based Food & Health web application** that helps users make better food choices, analyze nutrition, and build healthier eating habits using AI-driven insights and real-world nutrition data.

---

## 🚀 Live Features

🍽️ **Smart Meal Analysis**
- Breaks down calories, protein, fats, sugar, and nutrients
- Uses real nutrition databases (Open Food Facts + USDA)

⚠️ **Health Insights Engine**
- Detects unhealthy eating patterns
- Identifies skipped meals, high sugar intake, low protein diet

🔄 **Smart Food Alternatives**
- Suggests healthier food replacements
- Context-aware recommendations based on user goals

💡 **Habit Building Coach**
- Daily tips for hydration, portion control, and consistency
- Personalized nudges based on behavior

📊 **User Personalization**
- Tracks meal history
- Adapts recommendations based on:
  - Health goals (weight loss, muscle gain, maintenance)
  - Mood and hunger input
  - Eating patterns
